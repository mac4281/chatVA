DETAILED ANSWER:

==========

RESPONSE:

There are six steps to authenticate a user in Encompass: 1. The administrator must grant access to Encompass LO Connect via the user's persona. 2. Using the new Access tab in the Persona settings, the administrator determines how a persona accesses Encompass. 3. The user must be assigned an access level for the persona. 4. The Encompass LO Connect Terms of Use must be reviewed in the Mobile Access Confirmation window. 5. The user must click "I understand and accept these terms" and click OK. 6. The administrator must click the Save icon to save the persona changes.
