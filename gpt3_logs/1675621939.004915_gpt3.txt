DETAILED ANSWER:

==========

RESPONSE:

There are twelve steps to authenticate a user in Encompass. First, the administrator must log into EncompassLOConnect as an administrator. From the Encompass Admin Portal, the administrator will select Company/User Setup. Next, the administrator will select Services Management. After selecting the service category, the administrator will select the Menu icon in the actions column for a Manual Service and then select Edit. On the Edit Service Setup page, the administrator will select New Credentials. On the Credentials page, the administrator will select New Credentials. The administrator will enter a unique name in the Credential Name field and a description in the Description field. In the Users section, the administrator will navigate through the organization hierarchy or the Users list to select the users who will use the credentials in this setup. To move to a folder in the Organization, the administrator will select the Down Arrow icon. To move a folder in the hierarchy, or an individual user to the Added Users column, the administrator will select the Move Arrow icon. To search for a specific user, the administrator will select the Search icon. In the Credentials section, the administrator will enter the credentials provided by the service vendor and then select Save. The administrator can add another set of credentials by selecting New Credentials and repeating steps 8-11. For OptimalBlue credentials, see the note below. For the Scenario Comparison Tools, the administrator must provide the Account ID and user name. The Account ID is provided by OptimalBlue and the user name is the specific username for the EncompassLOConnect user. A separate set of credentials must be set up for each user unless the user name is shared by multiple users. OptimalBlue credentials for Loans and Opportunities settings are available in the Encompass Admin portal. Using these settings, the Encompass administrator can enable access to OptimalBlue as a Product and Pricing vendor in EncompassLOConnect. Only a user with the Super Administrator persona can enable access to OptimalBlue.
