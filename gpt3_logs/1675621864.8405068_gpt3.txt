RESPONSE:

There are nine steps to authenticate a user in Encompass: 1. Log into Encompass LO Connect as an administrator. 2. Click Customization, and then click Custom Tools. 3. Select New Custom Tool. 4. On the New Custom Tool window, enter the name of the tool in the Name field. 5. Enter a description of the tool in the Description field. 6. Select Custom Tool from the Tool Type drop-down list. 7. Enter the web address where the custom tool is hosted in the URL field. Note that this web address must be a secure (https) address. 8. Select the device type that the tool will be available for in the Device Type field. 9. Select Save.
