RESPONSE:

There are nine steps to authenticate a user in Encompass: 1. From the Encompass Admin Portal, select Opportunities Setup. 2. Select Affordability Management. 3. On the Affordability Controls page, set the toggle to On to enable the controls or Off to disable them. 4. Set the maximum Housing Ratio and Debt Ratio limitations for each product type. The selection is saved after you enter the value. 5. From the Encompass Admin Portal, select Opportunities Setup. 6. Select Eligibility Letter Management. 7. To add a new eligibility letter, select Add . 8. Use the editor to create an eligibility letter that is used when an LOConnect user sends an eligibility letter. 9. To add a custom field to the letter, select Insert Custom Field <>. This option is used to automatically populate the message with specific fields from the scenario data. 10. Select a field name to insert into the letter, and then select Add. When the letter is sent, these fields display the details from the specific scenario. 11. When you are finished editing the letter, select Save. 12. To duplicate an existing eligibility letter select Duplicate . 13. To delete an existing template, select the template and then select Delete . 14. Set the toggle to On to enable eligibility letters for your users or Off to disable them. 15. From the Encompass Admin Portal, select Opportunities Setup. 16. Select Email Templates.
