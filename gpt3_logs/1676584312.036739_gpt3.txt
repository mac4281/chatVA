To submit loan documents to the VA, you can start a new loan or edit an existing loan from the Pipeline. When you start a new loan, you can enter the borrower and/or co-borrower information and then save the scenario or request a lock. If you are editing an existing loan, you can view the loan details and then save the scenario or request a lock.
